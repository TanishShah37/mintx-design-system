# Design Tokens

All Mintx components consume CSS custom properties defined in `tokens.css`. This ensures consistent light/dark mode switching, single-source-of-truth colour management, and zero JavaScript cost for theming.

---

## Semantic Tokens (CSS Variables)

These are the tokens components actually use — they adapt between light and dark mode automatically.

### Background Tokens

| Token | Light | Dark | Use |
|---|---|---|---|
| `--bg-base` | `#F5F8F7` | `#080E0D` | Page background |
| `--bg-surface` | `#FFFFFF` | `var(--bg-surface, #0D1614)` | Card, panel surfaces |
| `--bg-elevated` | `#FFFFFF` | `#141F1D` | Modals, dropdowns |
| `--bg-overlay` | `rgba(20,31,29,0.06)` | `rgba(255,255,255,0.05)` | Hover states, overlays |
| `--bg-brand-soft` | `var(--bg-brand-soft, #E6FBF5)` | `rgba(0,179,138,0.12)` | Active badge bg, soft CTA |

### Text Tokens

| Token | Light | Dark | Use |
|---|---|---|---|
| `--text-primary` | `#141F1D` | `var(--text-primary, #E8EFED)` | Headlines, strong labels |
| `--text-secondary` | `#3A524D` | `var(--text-secondary, #9BACA6)` | Body copy, descriptions |
| `--text-tertiary` | `var(--chart-neutral, var(--chart-neutral, #748A83))` | `var(--text-tertiary, #5A706A)` | Metadata, hints, disabled labels |
| `--text-brand` | `#008F6E` | `#3DDCBA` | Links, active nav, brand accent |

### Border Tokens

| Token | Light | Dark | Use |
|---|---|---|---|
| `--border-subtle` | `rgba(20,31,29,0.07)` | `rgba(232,239,237,0.06)` | Dividers, subtle separators |
| `--border-default` | `rgba(20,31,29,0.12)` | `rgba(232,239,237,0.10)` | Card borders, input borders |
| `--border-strong` | `rgba(20,31,29,0.22)` | `rgba(232,239,237,0.18)` | Focused inputs, strong dividers |

### Shadow Tokens

| Token | Value (Dark) | Use |
|---|---|---|
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.30)` | Cards, inputs |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.40)` | Dropdowns, raised elements |
| `--shadow-lg` | `0 12px 32px rgba(0,0,0,0.50)` | Modals, drawers |
| `--shadow-brand` | `0 4px 20px rgba(0,179,138,0.35)` | Primary CTAs, focus glow |

## Spacing Scale

4px base unit. All spacing tokens map to `sp-N` Tailwind utilities.

| Token | Value | Tailwind class |
|---|---|---|
| `--sp-1` | 4px | `p-sp-1`, `m-sp-1`, `gap-sp-1` |
| `--sp-2` | 8px | `p-sp-2` |
| `--sp-3` | 12px | `p-sp-3` |
| `--sp-4` | 16px | `p-sp-4` |
| `--sp-5` | 20px | `p-sp-5` |
| `--sp-6` | 24px | `p-sp-6` |
| `--sp-8` | 32px | `p-sp-8` |
| `--sp-10` | 40px | `p-sp-10` |

### Motion Tokens

| Token | Value | Use |
|---|---|---|
| `--ease-out` | `cubic-bezier(0.16,1,0.3,1)` | Standard UI transitions |
| `--ease-spring` | `cubic-bezier(0.34,1.56,0.64,1)` | Toggle thumb, spring confirms |
| `--dur-fast` | `150ms` | Hover states, immediate feedback |
| `--dur-base` | `250ms` | Standard transitions |
| `--dur-slow` | `300ms` | Page transitions, large reveals |
