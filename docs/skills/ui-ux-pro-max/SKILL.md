# UI/UX Pro Max Skill

Integrated design intelligence for professional-grade web and mobile applications.

## Core Guidelines

### 1. Accessibility (CRITICAL)
- Contrast: 4.5:1 for normal text.
- Focus states: Visible 2-4px rings.
- Touch targets: Minimum 44x44px.
- Alt text & Aria labels for all meaningful icons.

### 2. Interaction
- Spacing: 4px/8px incremental grid.
- Feedback: Visual feedback (pressed/hover) within 100ms.
- Buttons: Scale feedback (0.95–1.05) on press.

### 3. Motion
- Duration: 150-300ms for micro-interactions.
- Easing: Ease-out for enter, ease-in for exit.
- Performance: Use transform/opacity only.
